👁
Qwen3.5-4B has native vision (multimodal). It uses early-fusion training on multimodal tokens, cross-generational parity with Qwen3-VL, 262K context, and support for 201 languages. Use it when you need image input, long documents, or the newest training. Qwen3-4B-Instruct handles text-only assistant tasks more efficiently.

PromptHub Routing Guide

Task type Route to Why
Plan a multi-step workflow or decompose a project Planner Qwen3-4B-Thinking Needs exploration + structured output; thinking mode shines here
Explain code, write a PR summary, general Q&A Worker Qwen3-4B-Instruct Balanced quality/speed, non-thinking mode, clean output
Attach a screenshot or image for analysis Vision Qwen3.5-4B Only model in your stack with native vision (early-fusion multimodal)
Summarize / read a very long document (>16K tokens) Vision Qwen3.5-4B Native 262K context; better at long-doc comprehension
Quick prompt rewrite before sending to 4B Editor Qwen3-1.7B Fast, cheap; acts as a pre-filter to improve prompt quality
Simple factual question, quick yes/no check Spec Qwen3-0.6B Fastest possible answer; saves tokens for trivial questions
MCP tool call with internal reasoning before action Planner → Worker Thinking model decides; Instruct executes the specific action
Image in + structured response out Vision → Worker Vision extracts info; Instruct formats the final response
