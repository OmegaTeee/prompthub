# Qwen CLI Interactive Help

<https://qwenlm-qwen-code.mintlify.app/cli/overview>

## Commands

Launch an interactive CLI use `-p/--prompt for non-interactive mode`

| command                   | options                                                                            | query           |
| ------------------------- | ---------------------------------------------------------------------------------- | --------------- |
| qwen [query..]            | Launch Qwen Code CLI                                                               | [default]       |
| qwen mcp                  | Manage MCP servers                                                                 |                 |
| qwen extensions <command> | Manage Qwen Code extensions.                                                       |                 |
| qwen auth                 | Configure Qwen authentication with OpenRouter, Coding Plan, API Key, or Qwen-OAuth |                 |
| qwen hooks                | Manage Qwen Code hooks (use /hooks in interactive mode).                           | [aliases: hook] |
| qwen channel              | Manage messaging channels (Telegram, Discord, etc.)                                |                 |
