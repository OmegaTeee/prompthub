## Qwen3 Model Family (4B, 1.7B, 0.6B)
- _Qwen3.5 4B_ is my main model for all my daily tasks
- _Qwen3 Thinking_ is my orchestrator agent
- _Qwen3 Instruct_ is my task assistant
- _Qwen3 1.7B_ is my quick response text rewrites
- _Qwen3 0.6B_ is my speculative decoding model for quick checks and token-saving

I use the Qwen3 models for everyday tasks such as code explanation, project planning, and PR summaries. The 4B models are my go-to choice for daily work. I use 0.6B for speculative decoding in LM Studio to support the 4B models. I use 1.7B for simple prompt edits and quick text rewrites before asking a 4B model for a more detailed response. The 0.6B model is also useful for quick checks and for saving tokens on simple questions.
