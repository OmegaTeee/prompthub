## Qwen3 Model Family (4B, 1.7B, 0.6B)
- _Qwen3.5 4B_ is my main model for all my daily tasks
- _Qwen3 Thinking_ is my orchstrator agent
- _Qwen3 Instruct_ is my task assistant
- _Qwen3 1.7B_ is my quick response text rewrites
- _Qwen3 0.6B_ is my speculative decoding model for quick checks and token-saving

I'm using the Qwen3 models for every day tasks such as code explanation, project planning, and PR summaries. The 4B models are my go-to all my daily tasks. Im using 0.6B for sepculive decoding in LM Studio in support of the 4B models. The 1.7B for simple prompt editor and quick text rewrites before I ask the 4B for a more detailed response. The 0.6B is also useful for quick checks and when I want to save tokens on a simple question.
