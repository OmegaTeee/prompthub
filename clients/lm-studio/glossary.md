# Large Language Model (LLM) Terms

---

## What is "streaming" in LLM context?
Streaming means generating text character-by-character (or token-by-token) as the model responds — instead of waiting to receive the full output before showing anything. This creates a real-time, conversational experience, like chat apps.

## What is "tool use" in LLM context?
Tool use refers to an LLM's ability to call external functions, APIs, or tools during its response generation. This allows the model to perform actions, retrieve information, or execute tasks beyond just generating text. For example, an LLM could use a calculator API to perform math or a web search API to get current information.

## What is "reasoning content" in LLM context?
Reasoning content is the internal thought process or rationale that an LLM generates while formulating a response. It can include the model's step-by-step logic, considerations, or any intermediate conclusions it reaches before arriving at the final answer. This content is often used to provide transparency into how the model arrives at its conclusions, helping users understand the reasoning behind the generated output.
